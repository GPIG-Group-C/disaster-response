Implementation

Use @mixin accordion() to apply the base styles necessary to make the accordion work. Then initialize the accordion with js.

By default, the CSS mixin is super barebones, but you can pass the value 'base' to the mixin to apply some base styles. You can always add your own presets to the mixin.

Also make sure to add data-block to your accordion with the name used for your block. This data-attribute is used as a naming hook for js.

Things to Note

Because the accordion works by adjusting the __content element's height, no vertical padding can be applied to the __content element. To help prevent any issues, I've added !important styles to prevent vertical padding. If you need control of spacing in the __content element, I recommend having all content wrapped in another div.

The style .js-accordion--open class is used to designate the open state. This class gets applied to 3 elements in the accordion block: __item, __title, & __content.
